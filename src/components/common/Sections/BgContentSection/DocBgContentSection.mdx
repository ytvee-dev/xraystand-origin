# Component: ContentSection

## Overview
`ContentSection` — это контейнер для отображения текстового блока с заголовком, описанием и дополнительным контентом.
Используется для построения секций страниц, где требуется выделить текстовую часть и вложенные элементы (например, карточки, изображения, блоки контента).

---

## Props

### `textData?`
Тип: `IContentSectionTextData`
Объект с текстовыми данными для секции.

Структура:
- **`title`**: `string` — основной заголовок (обязателен внутри объекта).
- **`description?`**: `string` — дополнительное описание.
- **`notificationLabel?`**: `string` — текст уведомления (не используется напрямую в текущей реализации, можно расширить).
- **`alertText?`**: `string` — дополнительный текст уведомления (резерв для расширений).
- **`colorScheme?`**: `{ background: string; text: string }` — цветовая схема секции:
- `background`: цвет фона секции.
- `text`: цвет текста заголовка.

### `children?`
Тип: `ReactNode`
Дополнительный JSX-контент, который отображается в нижней части секции.

---

## Usage

```tsx
import ContentSection from "@components/common/ContentSection";

export default function Example() {
  return (
    <ContentSection
      textData={{
        title: "О нас",
        description: "Мы создаем современные веб-приложения",
        colorScheme: {
          background: "#F5F5F5",
          text: "#333333",
        },
      }}
    >
      <p>Здесь может быть ваш контент: карточки, картинки или список.</p>
    </ContentSection>
  );
}
