# Component: LargeCard

## Overview
`LargeCard` — это компонент карточки крупного формата с заголовком, описанием, дополнительным контентом и встроенным уведомлением внизу. Используется для выделения ключевой информации, предоставления расширенного контента и информирования пользователя через нотификацию.

---

## Props

| Prop                 | Type                                                                 | Default            | Description |
|----------------------|----------------------------------------------------------------------|--------------------|-------------|
| `title`              | `string`                                                            | `""`               | Заголовок карточки. |
| `description`        | `string \| { subtitle: string; content: string[] }`                 | `""`               | Описание карточки. Может быть простой строкой или объектом с подзаголовком и массивом содержимого. |
| `children`           | `ReactElement \| null`                                              | `null`             | Дополнительный вложенный контент, рендерится в теле карточки. |
| `notificationLabel`  | `string`                                                            | `""`               | Текст уведомления, отображаемого внизу карточки. |
| `colorScheme`        | `ILargeCardColorScheme`                                             | `defaultColorScheme` | Цветовая схема для заголовка, подзаголовка, описания и уведомления. |

---

## Color Scheme (ILargeCardColorScheme)

| Key                           | Type     | Default  | Description |
|-------------------------------|----------|----------|-------------|
| `titleColor`                  | `string` | `#289FF5` | Цвет заголовка. |
| `subtitleColor`               | `string` | `black`  | Цвет подзаголовка/описания. |
| `descriptionColor`            | `string` | `black`  | Цвет текста описания. |
| `notificationBackgroundColor` | `string` | `beige`  | Фон уведомления. |
| `notificationTextColor`       | `string` | `black`  | Цвет текста уведомления. |
| `notificationIconColor`       | `string` | `beige`  | Цвет иконки уведомления. |

---

## Usage

```tsx
import LargeCard from "@components/common/Cards/LargeCard";

export default function Example() {
  return (
    <LargeCard
      title="Pro Plan"
      description="Лучший выбор для профессионалов"
      notificationLabel="Скидка 20% до конца месяца"
      colorScheme={{
        titleColor: "#222",
        subtitleColor: "#666",
        notificationBackgroundColor: "#F5F5F5",
        notificationTextColor: "#111",
        notificationIconColor: "#289FF5",
      }}
    >
      <div>
        <p>Включает расширенные метрики, приоритетную поддержку и доступ к бета-функциям.</p>
      </div>
    </LargeCard>
  );
}
