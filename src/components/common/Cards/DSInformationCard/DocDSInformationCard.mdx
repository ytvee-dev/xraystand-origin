# Component: DSInformationCard

## Overview
Информационная карточка со списком элементов. Может содержать заголовок и список опций (`liOptions`) с различными маркерами. Используется для структурированного отображения пунктов/фактов.

---

## Props

| Prop              | Type                                  | Default                | Description |
|-------------------|---------------------------------------|------------------------|-------------|
| `marker`          | `"decimal" \| "disc" \| "circle" \| "square" \| "none"` | `"decimal"` | Тип маркера для списка. |
| `title`           | `string`                              | `""`                   | Заголовок карточки. |
| `liOptions`       | `string[]`                            | `[]`                   | Список строковых элементов, которые выводятся в `<ul>`. |
| `action`          | `() => void`                          | `() => {}`             | Обработчик клика по карточке (`CardActionArea`). |
| `backgroundColor` | `string`                              | `"#FFFFFF"`            | Цвет фона карточки. |
| `width`           | `string`                              | `"270px"`              | Максимальная ширина карточки (`XXpx` или `100%`). |
| `sxText`          | `{ titleColor?: string; titleFontSize?: string \| number; fontWeight?: number; labelColor?: string; labelFontSize?: string \| number }` | `defaultDSCardSX` | Кастомизация текста заголовка и списка. |
| `sxContent`       | `{ minHeight?: string; minWidth?: string; maxWidth?: string; listPadding?: string }` | `defaultDSInformationCardSX` | Стили контейнера карточки. |

---

## Usage

```tsx
import DSInformationCard from "@components/common/Cards/DSInformationCard";
import { defaultDSCardSX, defaultDSInformationCardSX } from "@components/common/Cards/cardsMeta";

export default function Example() {
  return (
    <DSInformationCard
      title="Features"
      liOptions={["Fast setup", "Customizable", "Responsive design"]}
      marker="disc"
      backgroundColor="#FAFAFA"
      width="320px"
      action={() => console.log("Card clicked")}
      sxText={{
        ...defaultDSCardSX,
        titleColor: "#111",
        labelColor: "#444",
        labelFontSize: "14px",
      }}
      sxContent={{
        ...defaultDSInformationCardSX,
        minHeight: "200px",
        listPadding: "20px",
      }}
    />
  );
}
