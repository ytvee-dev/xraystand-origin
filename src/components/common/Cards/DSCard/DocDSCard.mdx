# DSCard Component Documentation

`DSCard` — это универсальная карточка на базе Material‑UI, которая может отображать изображение, заголовок, подпись и кнопку‑ссылку.

## Импорт

```tsx
import DSCard from '@components/common/Cards/DSCard';
```

```ts
export const dsCardImagePathPrefix = '/static/images/cards/';
export const defaultDSCardSX = {
  imgObjectFit: 'cover',
  imgPadding: 0,
  titleColor: '#000',
  labelColor: '#666',
  linkColor: '#1976d2',
};
```

---

## Props

| Свойство          | Тип                               | По умолчанию           | Описание                                                                                                                                                                                                                                                   |
| ----------------- | --------------------------------- | ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `imageName`       | `string`                          | `""`                   | Имя файла картинки. Если пустая строка, изображение не показывается.                                                                                                                                                                                       |
| `title`           | `string`                          | `""`                   | Заголовок карточки. Если пустая строка, заголовок не отображается.                                                                                                                                                                                         |
| `label`           | `string`                          | `""`                   | Подпись/описание под заголовком.                                                                                                                                                                                                                           |
| `linkedText`      | `string`                          | `""`                   | Текст кнопки‑ссылки. Если пустая строка, кнопка не отображается.                                                                                                                                                                                           |
| `action`          | `() => void`                      | `() => alert('click')` | Обработчик клика по всей карточке и кнопке.                                                                                                                                                                                                                |
| `backgroundColor` | `string`                          | `"#FFFFFF"`            | Цвет фона карточки (любое валидное CSS‑значение).                                                                                                                                                                                                          |
| `width`           | `string`                          | `"270px"`              | Максимальная ширина карточки (`XXpx` или `100%`).                                                                                                                                                                                                          |
| `sxText`          | `Partial<typeof defaultDSCardSX>` | `defaultDSCardSX`      | Объект стилей для текста и изображения:<br />- `imgObjectFit` — `object-fit` для `<img />`<br />- `imgPadding` — отступ вокруг изображения<br />- `titleColor` — цвет заголовка<br />- `labelColor` — цвет подписи<br />- `linkColor` — цвет текста кнопки |

## Пример использования

```tsx
import React from 'react';
import DSCard from '@components/common/Cards/DSCard';

const Example = () => {
  const handleClick = () => console.log('Карточка нажата');

  return (
    <DSCard
      imageName="product123.png"
      title="Новый продукт"
      label="Описание продукта и его особенности"
      linkedText="Узнать подробнее"
      action={handleClick}
      backgroundColor="#FAFAFA"
      width="300px"
      sxText={{
        imgObjectFit: 'contain',
        imgPadding: '8px',
        titleColor: '#333333',
        labelColor: '#555555',
        linkColor: '#FF5722',
      }}
    />
  );
};

export default Example;
```

## Рекомендации по стилизации

1. **Изображение**

* По умолчанию высота изображения равна ширине: `height={width}`.
* Для сохранения пропорций используйте `sxText.imgObjectFit: 'contain'` или другое.

2. **Ширина**

* Фиксированная (`"270px"`, `"300px"`) или адаптивная (`"100%"`).

3. **Цвета текста**

* Переопределяйте через `sxText`, остальные значения подтянутся из `defaultDSCardSX`.

4. **Обработчики**

* И карточка, и кнопка используют один `action`. Для разных действий нужно менять код компонента.
