# Component: DSCardsWrapper

## Overview
`DSCardsWrapper` — это обертка для карточек, которая управляет их расположением и адаптацией под ширину экрана.
Позволяет настраивать ширину контейнера, поведение переноса (`wrap`/`nowrap`), отступы между карточками и скролл.

---

## Props

### `screenMaxWidth`
Тип: `number`
Значение по умолчанию: `1000`
Ширина экрана (в px), при которой карточки переходят в режим переноса (`wrap`).

---

### `wrapperMaxWidth`
Тип: `string`
Значение по умолчанию: `"500px"`
Максимальная ширина контейнера карточек.

---

### `maxHeight`
Тип: `string`
Значение по умолчанию: `""`
Максимальная высота контейнера. Если не указано — не ограничивается.

---

### `cardsGap`
Тип: `string`
Значение по умолчанию: `"16px"`
Расстояние между карточками (`gap`).

---

### `overflow`
Тип: `"auto" | "hidden" | "scroll" | "visible" | "none"`
Значение по умолчанию: `"none"`
Задает CSS-свойство `overflow` для контейнера.

---

### `isWrap`
Тип: `boolean`
Значение по умолчанию: `false`
Флаг, принудительно включающий перенос карточек (`flex-wrap: wrap`).
Если `false`, то перенос включается автоматически при уменьшении ширины экрана ниже `screenMaxWidth`.

---

### `children`
Тип: `ReactNode`
Вложенные элементы (обычно карточки).

---

## Usage

```tsx
import DSCardsWrapper from "@components/common/Wrappers/DSCardsWrapper";
import DSCard from "@components/common/Cards/DSCard";

export default function Example() {
  return (
    <DSCardsWrapper wrapperMaxWidth="800px" cardsGap="24px">
      <DSCard title="Карточка 1" label="Описание 1" />
      <DSCard title="Карточка 2" label="Описание 2" />
      <DSCard title="Карточка 3" label="Описание 3" />
    </DSCardsWrapper>
  );
}
