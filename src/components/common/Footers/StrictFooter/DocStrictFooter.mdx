# Component: Footer (StrictFooter)

## Overview
`Footer` (или `StrictFooter`) — это настраиваемый компонент подвала сайта, который отображает логотип, описание, ссылки, контактную информацию, копирайт и блок "Made by".
Поддерживает светлую и тёмную цветовые схемы для логотипа и метки "Made by".

---

## Props

### `meta` (обязательный)
Тип: `IStrictFooterContent`

Содержит всю контентную информацию для футера:
- `logoPath`: `string` — путь к svg-спрайту логотипа.
- `content`:
- `description`: `string` — описание компании/проекта.
- `buttonsText`: `{ support: string; privacy: string; }` — подписи для кнопок "Поддержка" и "Политика".
- `madeBy`: `{ imagePath: { light: string; dark: string; }, link: string }` — информация о блоке "Made by".
- `contacts`: `{ email: string; phone: string }` — контакты для связи.
- `copyright`: `string` — текст копирайта.

---

### `sx` (опциональный)
Тип: `IStrictFooterSX`

| Prop               | Type     | Default     | Description |
|--------------------|----------|-------------|-------------|
| `backgroundColor`  | `string` | `"#181818"` | Цвет фона футера. |
| `boldTextColor`    | `string` | `"#ffffff"` | Цвет заголовков и акцентных элементов. |
| `textColor`        | `string` | `"#FFFFFF66"` | Цвет описания и дополнительного текста. |
| `copyrightColor`   | `string` | `"#FFFFFF66"` | Цвет текста копирайта. |
| `logoColorMode`    | `"light" \| "dark"` | `"light"` | Выбор логотипа в светлой или тёмной версии. |
| `madeByColorMode`  | `"light" \| "dark"` | `"light"` | Выбор изображения "Made by" для светлой или тёмной темы. |

---

## Usage

```tsx
import Footer from "@components/common/Footers/StrictFooter";

const footerMeta = {
  logoPath: "/assets/icons/",
  content: {
    description: "Наша компания создает инновационные решения для бизнеса.",
    buttonsText: {
      support: "Поддержка",
      privacy: "Политика конфиденциальности",
    },
    madeBy: {
      imagePath: {
        light: "/assets/madeby-light.png",
        dark: "/assets/madeby-dark.png",
      },
      link: "https://example.com",
    },
  },
  contacts: {
    email: "info@example.com",
    phone: "+7 (999) 123-45-67",
  },
  copyright: "© 2025 Example. Все права защищены.",
};

export default function Example() {
  return <Footer meta={footerMeta} />;
}
