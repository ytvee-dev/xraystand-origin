# Component: Header

## Overview
`Header` — это верхняя часть приложения, которая отображает логотип и, при необходимости, переключатель языков.
Используется для брендирования и настройки мультиязычности интерфейса.

---

## Props

### `singleLanguage` (обязательный)
Тип: `boolean`

- `true` — язык фиксированный, переключатель языков **не отображается**.
- `false` — отображается компонент `<LanguageSwitcher />`, позволяющий переключать язык интерфейса.

---

## Usage

```tsx
import Header from "@components/common/Header";

export default function Example() {
  return <Header singleLanguage={false} />;
}
